ToDiscord Extension User Manual

  • Use of Extension:

    • The extension will share content based on the following:
      • Share feature is used when a text is selected in the current tab
        • Only the selected text will be shared to the selected channel
      • Share feature is used when cursor/mouse/pointer is on a hyperlink
        • The URL will be shared to the selected channel
      • Share feature is used when cursor/mouse/pointer is on a browser tab
        Or nothing is selected and the cursor is pointing to the background of the web page
        • The URL of the current page will be shared to the selected channel
    • When sharing content, For a first-time user or when there are no channels added:
      • Instead of "To Discord", you'll see "Add Channel" option.

  • Extension settings:

    • Add a name and channel URL in the respective fields and hit "Save" to Save the Discord channel.
      • Note: when you hit "Save", the settings/options will be a[pplied and the page will be closed.
    • Use "Add Discord Channel" to add multiple channels at once.
    • Remove name and URL from boxes to delete the saved channel.
    • "Clear all" button will remove all the channels at once.(Use it with caution).

  • Requirements:

    • There are no pre-added channels, At least one discord channel should be added before you can share content with it.
    • This extension can only send content to channels with registered webhooks.
    • Content can be shared with only one channel at a time.
    • Get a webhook URL for a discord channel.
      • If you are a member of a discord server, Ask your server admin to give a webhook URL
      • If you are admin of the Discord server follow the below steps:
        1. Go to "Server Settings"
        2. Select "Webhooks"
        3. Click on "Create Webhook"
        4. Add a name for the webhook and select channel in which the webhook post will be posted.
        5. Copy the webhook URL
          • This URL is public, so be cautious before sharing it with others.
        6. Click "Save"
    • Keep a unique name for each channel when adding, this will help provide more options when sharing the content.
    • Channel names are case sensitive, i.e. 'mychannel', 'Mychannel', 'MyChannel' and 'myChannel' are all different channel names.
    • If two different channels have the same name, the last added channel will be used for sharing content.
